# Laboratorio RAG — Retrieval Augmented Generation

Laboratorio de construcción de un sistema RAG usando Python, FAISS, LangChain y Groq (LLaMA 3).

## Tecnologías
- Python / Google Colab
- FAISS (base vectorial)
- HuggingFace Embeddings
- Groq API (LLaMA 3.3)
- LangChain

## Contenido
- `laboratorio_rag.ipynb` — Notebook completo con código y análisis
- Capturas de ejecución de las 3 consultas
